# Code of Conduct

This project follows a simple standard: be respectful, constructive, and evidence-oriented.

Good participation includes:

- assuming good intent,
- discussing technical disagreement with specifics,
- avoiding personal attacks,
- respecting maintainers' time,
- helping newcomers understand the architecture.

Unacceptable behavior includes harassment, threats, discrimination, or sustained disruptive conduct.

Maintainers may moderate issues, discussions, pull requests, and contributions to keep the project healthy.
